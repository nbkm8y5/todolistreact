# Todo List App
### Built with React

open terminal in Mac or Linux environment

Type:

```
git clone https://github.com/nbkm8y5/todolistreact.git

cd todolistreact

npm start
```

Windows environment in powershell

```
git clone https://github.com/nbkm8y5/todolistreact.git

cd todolistreact

npm start
```

Note: please have node installed in your computer and git]
